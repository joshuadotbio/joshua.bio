About this website

Designed and developed by yours truly, joshua.bio is produced using the Liquid templating language, the Jekyll static site generator, and plain HTML/CSS written lovingly from scratch. It has a PageSpeed rank of 100/100, and is hosted on the amazingly cheap NearlyFreeSpeech service. You can find the entire source code for this website on GitHub, and you are free to reuse and remix my *.scss files however you wish.

Privacy

This website does not use cookies and makes no attempts to track you. But that's only because there's no need. There are many valid reasons why a website would want to track visitors, and I fully support the ethical use of web trackers.
